# refactoryExamSimpleArraySum

- Problem Name : Simple Array Sum
- Author of Problem Name : shashank21j
- URL to Problem : [Problem]
- Pseudocode code :
```
 function simpleArraySum(n, ar) {
        set result =0;
        for (set i =0; i < n; i++) {
        result += ar[i];    
        }
        return result;
 }
 function main() {
    set result = simpleArraySum(n, ar);
    print(result);
  }
```
